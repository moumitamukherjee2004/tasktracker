# Task Tracker ✅

A simple React-based Task Tracker application that helps you organize tasks with add, complete, delete, filter, and due date features. Tasks are stored in localStorage, so they persist even after refreshing the page.

---

## 🚀 Features
- Add tasks with optional due dates
- Mark tasks as completed (strike-through style)
- Delete tasks instantly
- Filter tasks: **All / Completed / Pending**
- Responsive design with clean UI
- LocalStorage persistence

---

## 🛠️ Tech Stack
- **React** (Hooks: useState, useEffect)
- **CSS** for styling
- **LocalStorage** for persistence
